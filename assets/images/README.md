# Assets Directory

Place your app assets here.

## Required Assets

### App Logo
- `assets/images/logo.png` — App logo (512×512 px recommended)
- `assets/images/logo_white.png` — White version for dark backgrounds

### Placeholder Images
- `assets/images/placeholder_tool.png` — Default tool image when no photo is available
- `assets/images/placeholder_user.png` — Default user avatar

### Onboarding (Optional)
- `assets/images/onboarding_1.png`
- `assets/images/onboarding_2.png`
- `assets/images/onboarding_3.png`

## Notes
- All images should be in PNG or JPEG format
- Keep sizes reasonable for mobile (< 500KB each)
- Use 2x and 3x variants for high-DPI screens:
  - `assets/images/2.0x/logo.png`
  - `assets/images/3.0x/logo.png`
